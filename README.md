# Carbon-UI-Design# NFTme

A modern, responsive NFT marketplace landing page built using **HTML, CSS**.  
Designed with a premium UI inspired by Figma, featuring glassmorphism, gradients, and smooth layout structure.

---

##  Features

-> Fully responsive design (Desktop, Tablet, Mobile)  
-> Premium UI (Glassmorphism + Gradient effects)  
-> NFT Card Stack Design (Hero section)  
-> Stats & Artwork sections  
-> Trending NFT Layout (Masonry-style)  
-> Popular Artists section  
-> Community CTA section  
-> Multi-page navigation:
- Home
- About Us
- Store  

---

##  Tech Stack

- HTML5  
- CSS3   
- Google Fonts

---

---

## Responsive Design

The website is optimized for:

-  Desktop (≥ 1024px)
-  Tablet (768px – 1024px)
-  Mobile (≤ 768px)

---

## Key UI Highlights

- Glass effect
- Gradient buttons and backgrounds
- Pixel-perfect spacing based on Figma
- Clean typography system
